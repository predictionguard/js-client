Completion

  • Completion(model, maxTokens, temperature, prompt): Promise<[Completion, null | Error]>

  • Chat generates chat completions based on the provided input.

    Parameters

    • model: completion.Model

      model represents the model to use for the request.

    • maxTokens: number

      maxTokens represents the maximum number of tokens in the generated chat.

    • temperature: number

      temperature represents the parameter for controlling randomness in generated chat.

    • prompt: string

    Returns Promise<[Completion, null | Error]>

    • A Promise with a Completion object and a client.Error object if the error is not null.

    Example

    import * as pg from 'predictionguard';

    const client = new pg.completion.Client('https://api.predictionguard.com', process.env.PGKEY);

    async function Completions() {
        const model = pg.completion.Model.NeuralChat7B;
        const prompt = 'Will I lose my hair';

        var [result, err] = await client.Completion(model, 1000, 1.0, prompt);
        if (err != null) {
            console.log('ERROR:' + err.error);
            return;
        }

        console.log('RESULT:' + result.choices[0].text);
    }
